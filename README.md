# Webbshop Next.js

#### Projektet innehåller:

- Next.js
- React
- TypeScript
- MUI
- NanoId
- Zod
- React-hook-form

## Länkar:

- [Next.js](https://nextjs.org/)
- [Material-UI](https://mui.com/)
- [Zod](https://zod.dev/)

- nmp install
- npm run dev
